# SistemaNotas
Sistema de notas con NodeJs y VueJs, 

## Se utiliza:
- middlewares 

librerias como: 
- bcrypt
- Underscore
- json webtoken

Con los cuales se puede verificar:
- Sesion correcta al sistema 
- El rol de usuario
- Agregar Middleware a Notas, para evitar que no se puedan modificar o eliminar si no esta identificado
